**Основная программа для тестирования ПИД-регулятора**  
Программа демонстрирует работу ПИД-регулятора на различных моделях объекта управления.

- **simulateSystem**  
  Проводит simulation системы с ПИД-регулятором

  Основная функция программы

### pid.h / pid.cpp
**Заголовочный файл для класса ПИД-регулятора**  
Класс, реализующий дискретный ПИД-регулятор с использованием рекуррентного алгоритма.

#### Класс `PIDController`

- **Атрибуты:**
  - `K` — Коэффициент передачи  
  - `T` — Постоянная интегрирования  
  - `Td` — Постоянная дифференцирования  
  - `T0` — Время квантования  
  - `q0, q1, q2` — Коэффициенты регулятора  
  - `prev_error` — Предыдущие ошибки e(k-1), e(k-2)  
  - `prev_output` — Предыдущее выходное значение u(k-1)

- **Методы:**
  - `PIDController(double K, double T, double Td, double T0)`  
    Конструктор ПИД-регулятора  
    - `K` — коэффициент передачи  
    - `T` — постоянная интегрирования  
    - `Td` — постоянная дифференцирования  
    - `T0` — время квантования  

  - `double calculate(double setpoint, double current_value)`  
    Вычисляет управляющее воздействие  
    - `setpoint` — заданное значение  
    - `current_value` — текущее значение процесса  
    - **Возвращает:** управляющее воздействие

  - `void reset()`  
    Сбрасывает состояние регулятора

  - `std::vector<double> getCoefficients() const`  
    Возвращает коэффициенты регулятора `[q0, q1, q2]`

### model.h / model.cpp
**Заголовочный файл для модели объекта управления**  
Класс, моделирующий объект управления, который может работать как в линейном, так и в нелинейном режиме.

#### Класс `ProcessModel`

- **Атрибуты:**
  - `params` — параметры модели  
  - `prev_value` — предыдущее значение выхода  

- **Методы:**
  - `ProcessModel(const std::vector<double>& params, double initial_value)`  
    Конструктор модели процесса  
    - `params` — вектор параметров модели  
    - `initial_value` — начальное значение  

  - `double linearModel(double input)`  
    Линейная модель объекта  
    - `input` — входное воздействие  
    - **Возвращает:** выходное значение  

  - `double nonlinearModel(double input)`  
    Нелинейная модель объекта  
    - `input` — входное воздействие  
    - **Возвращает:** выходное значение  

  - `void setInitialValue(double value)`  
    Устанавливает начальное значение
