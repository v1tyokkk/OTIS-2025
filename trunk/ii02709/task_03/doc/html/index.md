### Цель работы

Разработать на языке C++ программу, моделирующую работу дискретного ПИД-регулятора в **рекуррентной (инкрементальной) форме**:
u(k) = u(k−1) + q₀·e(k) + q₁·e(k−1) + q₂·e(k−2)
textс автоматическим расчётом коэффициентов q₀, q₁, q₂ по заданным непрерывным параметрам K, Ti, Td и периоду дискретизации T₀.

В качестве объекта управления использовать линейную и нелинейную модели первого порядка:

- Линейная:  
  y(k+1) = a·y(k) + b·u(k) + c

- Нелинейная (с насыщением по входу):  
  y(k+1) = a·y(k) + b·(u / (1 + α·|u|)) + c
