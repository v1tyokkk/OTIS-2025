\chapter{Документация ПИД-\/регулятора }

Программа реализует ПИД-\/регулятор для управления нелинейным объектом (температурой). Используется дискретная форма ПИД-\/регулятора и нелинейная модель объекта.\hypertarget{index_algorithm}{}\doxysection{\texorpdfstring{Алгоритм работы}{Алгоритм работы}}\label{index_algorithm}

\begin{DoxyEnumerate}
\item Ввод параметров модели объекта и ПИД-\/регулятора
\item Расчет коэффициентов ПИД-\/регулятора в дискретной форме
\item Циклическое моделирование системы управления
\item Расчет ошибки, управляющего воздействия и новой температуры
\item Вывод результатов моделирования 
\end{DoxyEnumerate}
