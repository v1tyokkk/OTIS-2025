## Detailed Description

Реализация линейной динамической модели объекта управления. 

Линейная модель описывает поведение объекта по разностному уравнению:   \[    y(k+1) = a \cdot y(k) + b \cdot u(k)
\]

где:

* ( y(k) ) — текущее состояние объекта,
* ( u(k) ) — управляющее воздействие,
* ( a ) — коэффициент сохранения предыдущего состояния,
* ( b ) — коэффициент влияния управляющего сигнала.
