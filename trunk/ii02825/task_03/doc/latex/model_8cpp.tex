\doxysubsection{Detailed Description}
Реализация линейной динамической модели объекта управления. 

Линейная модель описывает поведение объекта по разностному уравнению\+:   \[    y(k+1) = a \cdot y(k) + b \cdot u(k)
\]

где\+:
\begin{DoxyItemize}
\item ( y(k) ) — текущее состояние объекта,
\item ( u(k) ) — управляющее воздействие,
\item ( a ) — коэффициент сохранения предыдущего состояния,
\item ( b ) — коэффициент влияния управляющего сигнала. 
\end{DoxyItemize}
